# Mini-LMS-project

Мини-система обучения (LMS), написанная на чистом PHP без фреймворков.  
Проект демонстрирует работу с ролями, постами, оценками и базовой архитектурой MVC-подхода.

## Функционал

### Авторизация
- Регистрация / вход
- Хранение пользователей в MySQL
- Роли:
  - `student`
  - `teacher`
  - `guest`

### Посты
- Создание / редактирование / удаление
- Авторизация владельца поста
- Ограничение доступа к редактированию (только автор)
- Видимость постов по ролям
- Отображение автора и даты

### Дневник
#### Student:
- просмотр своих оценок

#### Teacher:
- выставление оценок студентам
- выбор предмета
- доступ только к своим предметам (teacher_subjects)

### Предметы учителя
- привязка предметов к teacher
- контроль доступа при выставлении оценок

### API
- `?page=api_posts`
- получение постов через fetch

## Структура БД

- `users`
- `posts`
- `grades`
- `teacher_subjects`

## Безопасность

- password_hash / password_verify
- CSRF защита (для форм)
- контроль доступа по ролям
- server-side проверка прав (не только UI)

## ⚙️ Установка

1. Установить XAMPP
2. Создать БД:
```sql
CREATE DATABASE demo;
```
3. Импортировать:
database.sql
4. Настроить подключение:
config/db.php
5. Запуск
```http://localhost/?page=home```
## Особенности проекта
- Vanilla PHP (без фреймворков)
- Простая MVC-структура
- Role-based access control
- Мини LMS логика (ученики / учителя / оценки)
- API + fetch интеграция
